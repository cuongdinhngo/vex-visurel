## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ or Bun runtime
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/cuongdinhngo/vex-visurel.git
cd vex-visurel
```

2. **Install dependencies**
```bash
# Using Bun (recommended)
bun install

# Or using npm
npm install

# Or using pnpm
pnpm install

# Or using yarn
yarn install
```

3. **Start development server**
```bash
# Using Bun
bun run dev

# Or using npm
npm run dev

# Or using pnpm
pnpm dev

# Or using yarn
yarn dev
```

4. **Open your browser**
Navigate to `http://localhost:3000` to see the application.
